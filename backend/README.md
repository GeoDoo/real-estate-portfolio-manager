# Backend

## Setup

1. Install dependencies:
   ```sh
   ./venv/bin/pip install -r requirements.txt
   ```

2. Initialize the database:
   ```sh
   ./venv/bin/python run.py initdb
   ```

## Running

- Start the backend server:
  ```sh
  ./venv/bin/python run.py
  ```

- Run tests:
  ```sh
  ./venv/bin/python run.py test
  ```
